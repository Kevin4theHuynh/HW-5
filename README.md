# Day planner



This Day Planner allows you to log certain task at certain times. When loading the page you are presented with times starting at 9AM-5PM. Even after reloading the page your tasks will remain.

Technologies Used
- Bootstrap
- jQuery
- Moment.js

Languages Used
- HTML
- CSS
- JavaScript
